# auto_trade
自动交易币安刷alpha积分

操作视频在推特上查看。

1. 下载目录
2. 谷歌浏览器加载已解压的扩展程序
3. 在alpha的任何币交易页面就有显示了，必须是alpha板块的币，哪个链都支持。
比如页面：https://www.binance.com/zh-CN/alpha/bsc/0xc71b5f631354be6853efe9c3ab6b9590f8302e81
